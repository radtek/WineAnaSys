Instructions

How to create Windows icons

To create a Windows icon, design your icon using an image editor with the same dimensions as set in Icon Settings. If you are creating a 32x32 icon, the dimensions of the source image should be 32x32 pixels for best results. If the dimensions of the source image do not match the intended icon size, the image will be automatically re-sized when the icon is created. Once the source image is prepared, ensure the Save Mode is set to Windows Icon (*.ICO), then drag the image into the Imagicon window or select Open from the file menu. The source image can be in BMP, JPG or PNG format.

How to convert image formats

To convert an image to a different image format, select the destination image format in Save Mode, then drag the source image into the Imagicon window or select Open from the file menu. The source image can be in BMP, JPG or PNG format.

Settings

Icon Settings

16x16   32x32   48x48   64x64   128x128
Select Icon Settings from the options menu to set the icon dimensions in pixels. The recommended Windows icon sizes are 16x16 pixels, 32x32 pixels and 48x48 pixels. Custom sized icons with the dimensions of 64x64 pixels and 128x128 pixels are also available. All icons are encoded using 24-bit colour depth and alpha transparency bit masks. For best results, ensure the source image has the same dimensions as the icon size setting.

Enable Transparency
Select the Enable Transparency checkbox to enable alpha transparency of the icon image. When enabled, the Colour selection button will activate to allow the alpha mask colour to be set. All areas of the source image containing the alpha mask colour will be transparent in the final icon image. The JPG image format is not recommended as a source image when using this option, as it will not preserve the original alpha mask colour information.

Zero Encoded Alpha
Select the Zero Encoded Alpha checkbox to allow the transparent pixels of the icon to be encoded with zero RGB colour values. When disabled, the transparent pixels of the icon will be encoded using their original RGB colour values.

Image Transform

Use Slider   -   Use Pixel Dimensions
Select Image Transform from the options menu to set the transform operations performed on the image during the conversion process. The final image will be re-sized to the desired dimensions set with either the Slider or the Pixel Dimensions options. When the Slider is selected and set to 100%, no re-sizing will take place during the conversion process. The re-size settings do not apply to Windows icons.

Constrain Proportion
If the Pixel Dimensions are selected, a checkbox will activate giving the option to constrain the proportion of the final image. When enabled, only one dimension, either the image width (X) or the image height (Y) will need to be set. Imagicon will automatically calculate the remaining dimension to match the proportions of the source image.

Enable Bi-Linear Filtering
If the Slider is selected and set below 100%, or if the Pixel Dimensions are selected, a checkbox will activate giving the option to enable bi-linear filtering. When bi-linear filtering is active, the re-size process will smooth the appearance of the final image. When it is not active, the resulting image will have no smoothing effect applied. Images containing text will become unreadable when re-sized if bi-linear filtering is not enabled.

Horizontal Flip
Selecting Horizontal Flip will result in the final image or icon being mirrored in the horizontal plane.

Vertical Flip
Selecting Vertical Flip will result in the final image or icon being mirrored in the vertical plane.

Convert To Black & White
Selecting Convert To Black & White will result in the final image or icon being converted from colour into black and white (grey-scale). Note: the original colours can not be restored from a grey-scale image.

Save Mode

Windows or OS/2 Bitmap (*.BMP)
Independent JPEG Group (*.JPG)
Portable Network Graphics (*.PNG)
Windows Icon (*.ICO)
Select Save Mode from the options menu to set the destination image format. All source images processed by Imagicon will be converted into the format selected in the top combo box. BMP, JPG and PNG images are saved using 24-bit colour depth.

Superb Quality - Ratio (100:1)
High Quality - Ratio (75:1)
Medium Quality - Ratio (50:1)
Low Quality - Ratio (25:1)
When the JPG image format is selected in the top combo box, the lower combo box will activate giving the option to set the compression ratio for JPG files. These settings do not apply to BMP, PNG or ICO image formats.

Append Saved Filename
Select the Append Saved Filename checkbox to add additional text to all saved image filenames. Enter the required text into the text field provided below the checkbox.

Save Directory

Save Directory
Select Save Directory from the options menu to set the folder Imagicon uses to save the converted image files. Imagicon will save all image files into the same directory as the source file by default.

Customize Imagicon

Blue   Green   Red   Blue/Green   Blue/Purple   Custom
Select Customize Imagicon from the options menu to customize the image panel of the Imagicon window. A selection of coloured image panels are available, plus an option to choose a custom made image design. When creating custom panel images in an image editor, the dimensions of 200x150 pixels should be used. If the Custom image option is selected, the Select Custom Image button will be activated. Use the Select Custom Image button to locate and select your custom image design.

Select Text Colour
Use the Select Text Colour button to set the desired text colour in the image panel of the Imagicon window.

Enable Preview
Use the Enable Preview checkbox to enable or disable the preview of currently selected Customize Imagicon options.

Reset Settings

Reset Settings
Select Reset Settings from the options menu to restore all settings used by Imagicon back to default. This includes all menu, colour, GUI & save settings.
